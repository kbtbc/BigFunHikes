# Changelog - Trail Tales

## v1.0.0 - Initial Prototype (2026-01-19)

### Features
- Interactive Leaflet map with full Appalachian Trail route (312k GPS points)
- Beautiful homepage with hero section, stats overview, and latest entry preview
- Timeline page showing all journal entries in chronological order
- Individual entry detail pages with full content, photos, and daily maps
- Demo Day 1 journal entry: "Springer Mountain to Hawk Mountain Shelter"
  - 7.8 miles hiked, 1,200 ft elevation gain
  - Inspiring markdown content
  - Two beautiful trail photos
  - Sample GPX track

### Design System
- Custom color palette: forest greens, warm amber accents, earthy backgrounds
- Typography: Outfit (headings) + Inter (body)
- Sunset gradients and forest-inspired aesthetics
- Mobile-first responsive design
- Smooth animations and transitions

### Backend
- Complete REST API with Hono framework
- SQLite database with Prisma ORM
- Database models: User, JournalEntry, Photo
- API endpoints for CRUD operations on journal entries and photos
- Statistics endpoint for overall trail progress
- Better Auth integration for user authentication (email + OTP)
- Type-safe API contracts with Zod schemas

### Technical Stack
- Frontend: React 18 + Vite + TypeScript + Tailwind CSS
- Maps: Leaflet.js with OpenTopoMap terrain tiles
- Markdown: react-markdown with GitHub-flavored markdown support
- Backend: Bun + Hono + Prisma + SQLite
- Authentication: Better Auth
- Type safety: Zod schemas shared between frontend and backend

### Files Created
- `/webapp/src/components/`: Navbar, TrailMap, Stats, JournalEntry, Timeline
- `/webapp/src/pages/`: HomePage, TimelinePage, EntryDetailPage
- `/webapp/src/data/journalEntries.ts`: Mock demo data
- `/backend/src/routes/`: entries.ts, photos.ts, stats.ts
- `/backend/src/types.ts`: Shared Zod schemas
- `/backend/prisma/schema.prisma`: Database schema
- `/webapp/public/data/appalachian_trail.gpx`: Full AT route
- `/README.md`: Complete project documentation

### Data Source
- Appalachian Trail GPX file from TopoFusion (312,000 points, 4MB)
- OpenTopoMap tiles for beautiful outdoor terrain visualization

### Known Limitations
- Currently using mock data for demo (authentication not yet integrated on frontend)
- No offline mode yet
- No voice-to-text support yet
- No Google Drive sync yet

### Next Steps
- Integrate frontend with backend API
- Add user authentication flow on frontend
- Implement Google Drive sync for mobile publishing
- Add voice-to-text for journal entries
- Implement offline mode with service workers
- Add photo upload functionality
